首先知道反虚拟机的意义

例如，你有一个病毒，并且你把这个病毒发给了你的朋友，你的朋友不知道这个是不是病毒，于是使用虚拟机比如vmware打开了这个文件。

朋友看这虚拟机被捣毁的样子，又看着你急切的问你有没有打开的样子，陷入了沉默。。。

因此，为了避免这种尴尬的情况发生，你就不能让你的病毒在虚拟机中运行

所以这就是反虚拟机的意义

虚拟机和物理机的区别

虚拟机不是日常使用的电脑，所以肯定有很大的不同

例如，虚拟机有一些文件是物理机没有的

或者说，虚拟机的运行配置特别的低
