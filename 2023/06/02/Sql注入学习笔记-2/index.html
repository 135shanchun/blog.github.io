在 MySQL 中，UNION 是一种结合多个 SELECT 语句结果集的方法。该操作符将两个或多个 SELECT 语句的结果合并成一个结果集。

请注意以下几点：

  • 通过 UNION 连接的查询必须具有相同数量的列。
  • 列必须具有相似的数据类型。
  • 列的顺序必须相同。
  • UNION 消除了重复数据行，默认情况下是剔除
